# EasyAbp

An open source organization to enhance your ABP project development efficiency.

# ABP Framework

ABP Framework (opens new window) is an open-source web application framework based on ASP.NET Core (opens new window). All the products in EasyAbp organization are designed for the ABP framework.

# AbpHelper

AbpHelper provides code generation and more features to help you develop applications and modules with the ABP framework.

  • We provide AbpHelper.GUI (opens new window) to help you use it more visually.

  • You can also use AbpHelper.CLI (opens new window) directly if you are more used to command line.

# Modules

ABP is a modular application framework. We made a lot of useful and interesting modules, you can install them to your solution easily and use them for free.

# Introduction

# What is abp.io

abp.io is a modular application framework built on top of .net core. More information on the official abp website (opens new window).

# Open Source

abp.io is an open source project with its source code hosted on Github (opens new window).

# EasyAbp

An open source organization to enhance your ABP project development efficiency, More information on the EasyAbp.io (opens new window).

# Community

Get invloved with abp.io. Join the discussion at wechat.
